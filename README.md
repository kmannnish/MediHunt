# MediHunt- One stop solution for all your medical needs
Search medicines you need and get results from top 4 indian pharmacies.

To install:
```
$ pip install flask  
$ pip install selenium  
$ git clone "https://github.com/mihir-yadav/Medi_Hunt"  
$ python3 flaskRun.py  
```
